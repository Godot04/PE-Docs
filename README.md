# <span style="color:#AAAAFF">ProjectEcho-Docs</span> <span style="font-size: 0.5em;">сокращённо (PE-Docs)</span>

<div class="header" align="Left">

Репозиторий, созданный для удобного хранения документации по механикам, системам и т.п. проекта в <span style="color:#FFAAAA">Space Station 14</span> - <span style="color:#AAAAFF">Echo Protocol.</span>
Создан с целью упрощения поддержания механик, а так же создания лёгкой читаемости сложных механик и систем, что поможет как контрибьюторам так и форкам разобраться в работе отдельных систем, компонентов и т.п. и получить более детальное разъяснение по поводу условий и особенностей работы кодовой составляющей проекта.
